# Website (`packages/website`)

`remediation-website` — private, unpublished marketing/docs SPA.

## Stack

React 19 + react-router-dom 7 + Vite 6 (TypeScript). Deployed via Vercel
(`vercel.json`).

- Entry: `src/main.tsx` / `src/App.tsx`
- Hero terminal demo: `src/components/Terminal.tsx`
- Docs page: `src/pages/DocsPage.tsx`
- OG image generated by `scripts/generate-og-image.mjs` (`pnpm og`, uses
  `@resvg/resvg-js`)

## Brand / design constraints

`PRODUCT.md` is the source of truth. The site must "practice what it preaches" —
a design-system linter with impeccable visual hygiene. Personality:
**Precise · Opinionated · Unobtrusive.** The terminal demo is the hero; copy
frames what the user sees rather than selling. WCAG AA; reduced-motion degrades
gracefully.

Anti-references (avoid): Vercel/Stripe glassmorphism, fluffy SaaS pastels,
documentation-dump layouts.

The `impeccable` skill (`.agents/impeccable/`, `.claude/skills/impeccable`) is
available for UI work; past critiques are stored under `.impeccable/critique/`.
